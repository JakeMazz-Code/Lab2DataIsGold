Columbia Course Finder

Term: Fall 2025

A scraping demo that discovers subjects, scrapes the Directory of Classes, links recitations to their parent lectures, and presents a student-friendly search UI.

Pipeline

  1. Discover subjects for the chosen term (A–Z index)
  2. Scrape selected or ALL subjects (plain text listings)
  3. Parse sections (number, points, day/time, room, instructor)
  4. Link recitations (0 points or R##) to parent lectures
  5. Normalize schema → dashboard filters & visuals
